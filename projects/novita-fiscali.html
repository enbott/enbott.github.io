SUPSI Novita fiscali

A digital newspaper for SUPSI’s Department of Business and Economics, publishing monthly articles from students and professors, featuring user registration, premium content, automated Word import, and donation integration.

The project delivered an online journal for SUPSI (University of Applied Sciences and Arts of Southern Switzerland), aggregating tax and economic articles authored by students and professors into structured monthly issues. Built on Magnolia CMS—a content management system used to create and manage website pages—the platform enables public access to general content while requiring registration and login for premium or full-issue access. A custom importer automatically transforms submitted Microsoft Word documents into properly formatted articles within the CMS, preserving layout and metadata. Each monthly edition is curated as a cohesive publication. The site also includes a secure donation module powered by Datatrans to support ongoing editorial efforts.

Technologies used

  • Java 11 / Spring / Magnolia Blossom
  • HTML CSS Javascript
  • Freemarker (templating)
  • Apache POI (Word document parsing)
  • Datatrans Payment API
  • User authentication (Magnolia security)
  • MySQL / JCR
