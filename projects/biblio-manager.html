Multi-Tenant Library Management System

A scalable, multi-tenant platform for managing public library operations across multiple municipalities, handling loans, recalls, overdue notifications, teacher alerts, and reservations.

The objective was to design and implement a centralized library management system for our municipality, capable of supporting multiple independent tenants (libraries) within a single instance. The system efficiently tracks over 50,000 books, automates loan workflows, sends email recalls and overdue alerts, notifies teachers about student borrowings, and enables online reservations. Built with flexibility and scalability in mind, the platform was successfully adopted by three additional external municipalities, proving its adaptability to diverse administrative needs and operational scales.

Technologies used

  • PayloadCMS v.2
  • Node.js
  • React
  • MongoDB
  • Express.js
  • JWT (authentication)
  • Nodemailer (email notifications)
  • Cron scheduling
  • REST API
