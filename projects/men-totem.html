Interactive Municipal Map Totem for Mendrisio City Hall

A totem to find them all

A touchscreen-enabled web application displayed on a public totem, providing an interactive floor plan of the city hall with real-time pathfinding to selected offices.

The project involved developing a standalone, kiosk-mode web page deployed on a totem in the Mendrisio City Hall atrium. The interface features an interactive map of the municipal building, allowing visitors to select a destination office. Upon selection, the system automatically calculates and highlights the optimal route from the entrance to the chosen office, displayed clearly on the large screen. The map itself was modeled in 3D by me using Blender and exported as a high-quality SVG for seamless integration into the web application. The solution prioritizes usability for non-technical users, fast load times, and a responsive layout optimized for touch interaction in a public environment.

Technologies used

  • Vite, Tailwind CSS, HTML, CSS, TypeScript
  • Blender (3D modeling)
  • Mapplic
